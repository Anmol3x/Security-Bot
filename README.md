# Discord Security Bot

A comprehensive Discord security bot with anti-raid, anti-nuke, and moderation features.

## Features

- **Anti-Raid System**: Detect and prevent mass joins
- **Anti-Nuke Protection**: Protect against mass deletions and bans
- **Verification System**: CAPTCHA verification for new members
- **Moderation Tools**: Ban, kick, mute, warn, and more
- **Logging System**: Comprehensive event logging
- **Rate Limit Protection**: Prevent command spam
